카드 번호,타입,레어도,레벨,형태,속성,유형,DP,등장비용, 색1,색2,진화비용1,진화비용2,효과,진화원효과,시큐리티효과,입수정보

프로그램  개요
디지몬 카드 공식 사이트에서 웹페이지를 크롤링 하여 카드 정보를 수집한 후 API 상태로 저장
  1. 디지몬 카드 공식 홈페이지  웹 크롤링 
 2. 크롤링한  웹페이지 정보 Parsing 하여 JSON형태로 저장 

JSON에 저장되는 정보 
 CardNo: 카드 번호(디지몬 카드의 제품 번호, 동일 카드인지 식별하기 위해서 사용, 디지몬 카드는 카드의 동일판단을 카드번호를 통해 구분함  )
  CardType: 타입(카드의 종류 디지타마/디지몬/옵션카드로 분류 ),	
  CardRarity: 레어도(카드의 희소성을 판단하는 정도,C/U/R/SR/SEC정도가 있음
  CardLevel: 레벨(해당 카드의 레벨 표시)
  CardShape: 형태(해당 카드의 성장 단계 표시 ex)유년기,성장기, 성숙기)
  CardEntity: 속성( 해당카드 디지몬의 속성 표시 ex)바이러스,백신,데이터) 
  CardType: 유형(해당 카드 디지몬의 형태 정보 표시 ex)포유류형, 마왕형, 거조형)
 Card_DP: DP() (해당카드의 디지몬이 가지고 있는 라이프의 수치),
 Card_playCost: 등장비용(해당 카드를 플레이 할때 사용되는 코스트의 비용)
 CardColor1: 색1 (디지몬이 가지는 색 정보 1) 
 Card_Cost1: 색2(디지몬이 가지는 색 정보 2) 
 Card_EvolutionCost1: 진화비용1(특정색의 디지몬에서 해당디지몬으로 진화할수 있는 코스트1),
 Card_EvolutionCost2: 진화비용2(특정색의 디지몬에서 해당디지몬으로 진화할수 있는 코스트2),
 Card_Effect: 효과(등장 및 진화시 발휘 되는 디지몬의 효과에대한 정보나 옵션카드 발휘 시 사용되는 효과에 대한 정보)
 Card_Evo_Effect:진화원 효과 (위 카드의 디지몬이 진화했을때 얻게 되는 효과 ) 
 Card_Secu_Effect:시큐리티효과(위 카드가 시큐리티에서 체크될 경우에 발휘되는 효과)
 Card_Source: 입수정보(위 카드를 얻을수 있는 정보)
  Card_Image_url:위 카드의  웹이미지 주소
